# Book & Coffee Haven

## Overview

Book & Coffee Haven is a cozy space where book lovers and coffee enthusiasts can come together to enjoy their favorite reads while sipping on artisan coffee. This project showcases a simple yet elegant website that provides information about the café, its services, pricing plans, and a reservation form.

## Project Structure

- `index.html`: The main HTML document that contains the structure of the webpage.
- `style.css`: The stylesheet that defines the visual appearance of the website.
- `script.js`: The JavaScript file that adds interactivity to the webpage.
- `README.md`: This documentation file.

## Features

- Home section with a welcoming message and call-to-action to reserve a table.
- About section highlighting the unique offerings of the café.
- Services section detailing the menu items available.
- Pricing plans for different membership options.
- Customer reviews to showcase guest experiences.
- A reservation form for users to book a table.

## Getting Started

To get a local copy up and running, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/book-coffee-haven.git
   ```

2. Navigate to the project directory:
   ```
   cd book-coffee-haven
   ```

3. Open `index.html` in your web browser to view the site.

## Usage

- Explore the different sections of the website to learn more about the café.
- Use the reservation form to book a table for your visit.
